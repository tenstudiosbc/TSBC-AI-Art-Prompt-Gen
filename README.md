# TSBC AI Art Prompt Maker v2.0 🎨

> **Advanced AI Art Prompt Generation with Professional UI/UX**

## Overview
TSBC AI Art Prompt Maker is a sophisticated web-based tool designed to generate high-quality, detailed prompts for AI art generation. This professional-grade application assists artists, designers, content creators, and AI enthusiasts by providing comprehensive customization options and intelligent prompt building capabilities.

## ✨ Key Features

### 🎯 **Advanced Prompt Generation**
- **Smart Prompt Building** – Intelligent prompt construction with proper formatting
- **Negative Prompt Support** – Specify elements to avoid in generated images
- **Quality Tags** – Pre-defined quality enhancers for better AI results
- **Custom Parameters** – Add custom prefix/suffix text for personalization
- **Model Style Support** – Compatible with various AI model styles (Raw, Expressive, Cute, Scenic)

### 🎨 **Professional Interface**
- **Modern Glass-morphism Design** – Beautiful translucent interface with backdrop blur
- **Tabbed Navigation** – Organized into Character, Style & Effects, Scene & Composition, and Advanced sections
- **Progress Tracking** – Visual progress bar showing completion status
- **Responsive Layout** – Optimized for desktop, tablet, and mobile devices
- **Dark Theme** – Professional dark interface with gradient accents

### ⚡ **Quick Actions & Templates**
- **Pre-built Templates** – 6 professional templates for common art styles
- **Random Generation** – Smart random filling of all form fields
- **One-Click Operations** – Quick generate, clear all, and template application
- **Batch Processing** – Efficient workflow for multiple prompt generations

### 📚 **History & Favorites System**
- **Automatic History** – Saves last 20 generated prompts with timestamps
- **Favorites Management** – Save and organize your best prompt configurations
- **Quick Actions** – Load, copy, or delete history items with one click
- **Persistent Storage** – History saved to browser localStorage

### 🔧 **Advanced Tools**
- **Character Counter** – Real-time character counting with color-coded feedback
- **JSON Export** – Export complete settings and prompts as structured data
- **Multiple Copy Options** – Separate copy for main prompt and negative prompt
- **Form Validation** – Visual feedback for completion status

## 🚀 Quick Start

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tenstudiosbc/TSBC-AI-Art-Prompt-Gen.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TSBC-AI-Art-Prompt-Gen
   ```
3. Open `index.html` in your web browser or serve via a local server

### Usage Guide
1. **Choose a Tab** - Navigate between Character, Style & Effects, Scene & Composition, or Advanced
2. **Fill Options** - Select from dropdown menus or use Random Fill for quick generation
3. **Apply Templates** - Use pre-built templates for common art styles
4. **Generate Prompt** - Click the generate button to create your AI art prompt
5. **Copy & Use** - Copy the generated prompt and use it with your favorite AI art tool
